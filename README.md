新北市淡水區農會表示，只要有意願、有興趣耕種農作的民眾皆可承租農地，每次租期3年以上，以淡水區農地來說，不少人因參與農委會「園丁計劃訓練」來詢問出租農地，但最後因農地面積過大，讓只想有一小塊農地栽種的民眾卻步。 
 農地銀行網站待租的物件不少，住商不動產宜蘭羅東加盟店店長莊智清指出，農地若沒有在耕作，地主將會被課徵「空地稅」，因此地主都會盡量讓農地持續保持耕作。
